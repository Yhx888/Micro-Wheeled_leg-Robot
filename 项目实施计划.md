## 🔧 第二阶段：PCB 制作（第 2 周）

### 2.1 PCB 打样流程

```
1. 下载嘉立创 EDA（https://lceda.cn/）
2. 导入项目中的 PCB 源文件
3. 检查原理图和 PCB 布局
4. 生成 Gerber 文件
5. 在嘉立创下单（推荐 5 片起订）
6. 等待发货（通常 3-5 天）
```

### 2.2 需要制作的 PCB

| PCB 名称 | 功能 | 关键元件 | 焊接难度 |
|----------|------|----------|----------|
| **ControllerPCB** | 主控板 | ESP32 + L6234PD | ⭐⭐⭐⭐ |
| **EncoderPCB** | 编码器板 | AS5600 | ⭐⭐ |
| **IMUPCB** | IMU 板 | MPU6050 模块 | ⭐ |
| **ServoDebugPCB** | 舵机调试板 | 串口转接 | ⭐ |

### 2.3 焊接注意事项

#### L6234PD013TR 焊接要点
```
⚠️ 重要：淘宝上有很多假芯片，购买时注意辨别！

1. 检查芯片丝印是否清晰
2. 测试芯片引脚是否导通
3. 焊接温度：260°C 以内
4. 使用助焊剂辅助焊接
5. 焊接后检查连焊和虚焊
```

#### AS5600 焊接要点
```
1. 注意芯片方向（Pin 1 标记）
2. I2C 上拉电阻需要焊接
3. 编码器磁铁安装位置要准确
```

### 2.4 焊接顺序建议

```
1. 先焊接被动元件（电阻、电容）
2. 再焊接 IC 芯片
3. 最后焊接连接器
4. 每焊完一个模块就测试
```

## ⚖️ 第七阶段：平衡调参（第 6-7 周）

### 7.1 PID 参数说明

项目使用 LQR（线性二次调节器）进行平衡控制，分解为 4 个 PID 控制器：

```cpp
// 平衡控制参数
PIDController pid_angle     {.P = 1,    .I = 0,   .D = 0};  // 角度环
PIDController pid_gyro      {.P = 0.06, .I = 0,   .D = 0};  // 角速度环
PIDController pid_distance  {.P = 0.5,  .I = 0,   .D = 0};  // 位移环
PIDController pid_speed     {.P = 0.7,  .I = 0,   .D = 0};  // 速度环

// 转向控制参数
PIDController pid_yaw_angle {.P = 1.0,  .I = 0,   .D = 0};  // 偏航角环
PIDController pid_yaw_gyro  {.P = 0.04, .I = 0,   .D = 0};  // 偏航角速度环

// 腿部控制参数
PIDController pid_roll_angle{.P = 8,    .I = 0,   .D = 0};  // 横滚角环
```

### 7.2 调参步骤

#### Step 1: 角度环调参（pid_angle）
```
1. 将其他参数设为 0
2. 从小 P 值开始（如 0.5）
3. 逐渐增大 P 值
4. 观察机器人响应：
   - P 太小：机器人无力，无法站立
   - P 合适：机器人有站立趋势
   - P 太大：机器人剧烈抖动
5. 找到临界值，取 70% 作为最终值
```

#### Step 2: 角速度环调参（pid_gyro）
```
1. 保持 pid_angle 不变
2. 从小 P 值开始（如 0.02）
3. 逐渐增大 P 值
4. 观察阻尼效果：
   - P 太小：机器人摇摆不定
   - P 合适：机器人平稳站立
   - P 太大：机器人响应迟钝
5. 找到最佳阻尼效果
```

#### Step 3: 位移环调参（pid_distance）
```
1. 保持前两个参数不变
2. 从小 P 值开始（如 0.2）
3. 逐渐增大 P 值
4. 观察位置保持：
   - P 太小：机器人漂移
   - P 合适：机器人原地站立
   - P 太大：机器人频繁调整
```

#### Step 4: 速度环调参（pid_speed）
```
1. 保持前三个参数不变
2. 从小 P 值开始（如 0.3）
3. 逐渐增大 P 值
4. 观察速度响应：
   - P 太小：机器人运动迟缓
   - P 合适：机器人运动平滑
   - P 太大：机器人运动过猛
```

### 7.4 常见问题排查

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 机器人无法站立 | 角度 P 太小 | 增大 pid_angle.P |
| 机器人剧烈抖动 | 角度 P 太大 | 减小 pid_angle.P |
| 机器人摇摆不定 | 角速度 P 太小 | 增大 pid_gyro.P |
| 机器人漂移 | 位移 P 太小 | 增大 pid_distance.P |
| 机器人频繁调整 | 位移 P 太大 | 减小 pid_distance.P |
| 转向不灵敏 | 偏航 P 太小 | 增大 pid_yaw_angle.P |
| 转向过猛 | 偏航 P 太大 | 减小 pid_yaw_angle.P |

## ⚠️ 注意事项汇总

### 安全注意事项
```
1. 电池安全：不要过充过放，注意正负极
2. 焊接安全：通风环境，避免吸入焊烟
3. 机械安全：安装时小心夹手
4. 电气安全：首次上电串联限流电阻
```

### 质量注意事项
```
1. PCB 焊接质量：检查连焊虚焊
2. 机械精度：3D 打印件精度影响性能
3. 芯片真伪：L6234PD 假货多
4. 舵机标定：必须正确配置 ID 和行程
```

### 调试注意事项
```
1. 迭代调参：每次只调一个参数
2. 记录参数：调参前后记录数值
3. 安全第一：调参时有人扶着机器人
4. 耐心调试：平衡调参需要时间
```
